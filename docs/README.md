## Project scope at a glance

The application currently supports:

- interactive supply-chain visualization,
- live-style simulation of inventory movement,
- depot-level demand forecasting,
- and backend APIs that connect the UI to persistent data.

Use the docs in this folder to explain the project to technical users, managers, or evaluators.
